# File Extension Counter

A Python script that counts the number of files by their extensions in a given folder.

## Description
The program scans a user-specified folder and identifies file extensions.
It then displays how many files exist for each extension.

## Features
- Folder scanning
- File extension detection
- Clean and readable output
- Useful for organizing files

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Open terminal in the project directory
3. Run:
   python extension_counter.py
4. Enter a valid folder path
